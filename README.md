# etcdsimpan
etcdsimpan is simple tool for backup kubernetes etcd datastore

## How to use
 
1. Clone this repository to your local  
2. Install required requirements  

    On Linux  

        pip3 install -r requirements.txt  

    On Windows

        pip install -r requirements.txt  

3. Run main.py  

    On Linux  

        python3 main.py  

    On Windows  

        python main.py
